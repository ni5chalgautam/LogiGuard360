# LogiGuard360 Backend

## Quick start
1) Install Node.js (18+) and MongoDB (local or Atlas)
2) Copy `.env.example` -> `.env` and set values
3) Install deps and run:

```bash
npm install
npm start
```

Then open:
- http://localhost:5000  (frontend served by backend)
- API base: http://localhost:5000/api
